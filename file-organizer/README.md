# File Organizer

A lightweight Linux-friendly Python script for organizing files into folders by type.

## About This Project

This project is part of my journey learning Python automation and building useful command-line tools.

## Features

- Organizes files into folders
- Dry-run safety mode
- Live move mode
- Simple command line interface

## Installation

Clone the repo:

git clone https://github.com/yourname/python-automation-projects.git

## Usage

Dry run:

python3 file_organizer.py foldername

Live mode:

python3 file_organizer.py foldername --live
